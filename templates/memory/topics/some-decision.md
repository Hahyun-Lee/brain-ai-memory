# Example decision

- **Date:** YYYY-MM-DD
- **Status:** active
- **Decision:** State the chosen direction in one sentence.
- **Reason:** Record the constraint or evidence that determined the choice.
- **Revisit when:** Name the condition that would justify reconsolidating or
  superseding this decision.

When the decision changes, preserve any durable lesson, update the active
pointer, and remove the superseded version from the always-loaded index.
