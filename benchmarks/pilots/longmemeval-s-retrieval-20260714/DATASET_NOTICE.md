# LongMemEval dataset notice

The artifacts in this directory were derived from the cleaned LongMemEval-S
dataset published by Di Wu and collaborators. Some JSONL records contain
dataset excerpts needed to audit retrieval ranks. The dataset is published
under the MIT License.

Source: <https://huggingface.co/datasets/xiaowu0162/longmemeval-cleaned>

Paper: Di Wu, Hongwei Wang, Wenhao Yu, Yuwei Zhang, Kai-Wei Chang, and Dong Yu.
"LongMemEval: Benchmarking Chat Assistants on Long-Term Interactive Memory."
ICLR 2025. <https://arxiv.org/abs/2410.10813>
